# Track: ballanimeベンチマークのメモリ使用量表示機能強化

## 概要 (Overview)
このトラックの目的は、`uv run pi0disp ballanime` コマンドによって実行される既存のベンチマーク機能に、`ballanime` および `pigpiod` プロセスのメモリ使用量を計測し、ベンチマーク終了後にその平均値を出力する機能を追加することです。これにより、ユーザーはベンチマーク実行中のシステムリソース消費をより詳細に把握できるようになります。

## 機能要件 (Functional Requirements)
- `uv run pi0disp ballanime` のベンチマーク実行中に、`ballanime` プロセスと `pigpiod` プロセスのメモリ使用量を計測し、ベンチマーク終了後にその平均値を出力する。
- メモリ使用量は自動判別された適切な単位（例: MB, KB, バイト）で表示される。

## 非機能要件 (Non-Functional Requirements)
- ベンチマークの実行パフォーマンスに大きな影響を与えないこと。
- 既存のベンチマーク出力と統合され、視認性が高いこと。

## 受け入れ基準 (Acceptance Criteria)
- ベンチマーク終了時、`ballanime` プロセスと `pigpiod` プロセスの平均メモリ使用量が画面に出力されること。
- メモリ使用量の単位が自動的に最適なものに調整されて表示されること。
- ベンチマーク全体の実行時間に大きな遅延が発生しないこと。

## 範囲外 (Out of Scope)
- `ballanime` および `pigpiod` 以外のプロセスのメモリ使用量の表示。
- メモリ使用量の履歴データの保存やグラフ表示。
- メモリ使用量の閾値を超えた場合の警告機能。

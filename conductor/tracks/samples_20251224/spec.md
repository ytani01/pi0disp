# サンプルプログラムの作成 Spec

## 目的
既存のpi0dispライブラリの基本的な使い方を示すサンプルプログラムを開発し、ユーザーが容易にライブラリの機能を理解し、活用できるようにする。

## 範囲
以下の機能を持つサンプルプログラムを作成する。
*   **初期化**: `pi0disp` ライブラリと ST7789V ディスプレイの基本的な初期化処理
*   **描画**: `PIL.Image` オブジェクトを使用して、ディスプレイに画像や図形を描画する方法
*   **表示**: `ST7789V.display()` メソッドを使用して、描画した内容をディスプレイに表示する方法
*   **部分更新**: `ST7789V.display_region()` メソッドを使用して、画面の一部のみを効率的に更新する方法

## 技術要件
*   **言語**: Python
*   **ライブラリ**: `pi0disp`, `Pillow` (PIL)
*   **ハードウェア**: Raspberry Pi, ST7789Vディスプレイ

## 期待される成果
*   `samples/` ディレクトリ内に新しいサンプルプログラムファイルが追加される。
*   サンプルプログラムは、上記目的と範囲に記載された機能を網羅している。
*   コメントやドキュメンテーションが適切に記述され、初心者でも理解しやすい内容となっている。

# Track Specification: lcd-check subcommand

## Overview
`samples/lcd_check.py` を `pi0disp` のサブコマンド `lcd-check` として統合します。
このコマンドは、ST7789Vディスプレイの正しい `invert`（色反転）および `bgr`（色順序）の設定を視覚的に特定するためのテストパターンを表示します。

## Functional Requirements
- **サブコマンドの実装**: `uv run pi0disp lcd-check` として実行可能にする。
- **テストパターンの表示**:
    - 背景が黒、上から赤・緑・青の帯が表示されるテストパターンを描画する。
    - 現在のテストインデックス（例: 1/4）と、適用されている `invert`, `bgr` の値を画面上に描画する。
- **4つの組み合わせをテスト**:
    - `invert`, `bgr` のすべての組み合わせを順次表示する。
- **進行制御**:
    - デフォルトでは Enter キー入力を待機。`--wait` 指定時は自動進行。
- **コマンドラインオプション**:
    - `--rotation`, `--invert`, `--bgr`, `--wait` 等をサポート。

## Non-Functional Requirements
- **憶測の排除と事実に基づく検証**: 
    - 開発中はデバッグログを詳細に出力し、内部状態を可視化する。
    - 描画結果をキャプチャ（画像保存）し、意図通りのピクセル値になっているかプログラムで検証する。
- **自動テストの重視**: 物理デバイスがなくても、ロジックの正当性を画像比較やモックを用いて自動テストする。
- **既存アーキテクチャへの適合**: `src/pi0disp/commands/` 配下に Click コマンドとして実装。

## Acceptance Criteria
- [ ] `uv run pi0disp lcd-check` で実行でき、オプションが正しく解釈されること。
- [ ] 自動テストにより、各設定パターンにおいて正しい色（RGB）が正しい位置に描画されていることが検証されていること。
- [ ] `wait` オプションの動作が、モックタイムを用いたテストで検証されていること。
